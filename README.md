# Windows IT Support Home Lab

## Overview
Built a Windows-based IT support lab using VirtualBox to simulate common help desk scenarios.

## Technologies
- Windows 11
- VirtualBox
- TCP/IP
- DNS
- Event Viewer

## Tasks Performed
- User account creation and management
- Password resets
- Network troubleshooting
- DNS troubleshooting
- Software installation and removal
- Event log analysis

## Skills Demonstrated
- Windows Administration
- Technical Troubleshooting
- Customer Support
- Documentation
